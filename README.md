# Student Grade Management System 🎓

---

## 🚀 Features
- **Student Profile Creation:** Capture student names and unique IDs.
- **Dynamic Grade Storage:** Uses `List<float>` to store multiple subject grades.
- **Automated Calculations:** Built-in logic to calculate the average grade.
- **Data Validation:** Implements `int.TryParse` and `float.TryParse` with loops to ensure the program never crashes.

---

## 🛠️ Concepts Applied
- **Object-Oriented Programming (OOP):** Encapsulation through a dedicated `Student` class and constructor.
- **Control Flow:** Advanced use of `while` loops for input verification.
- **Collections:** Management of dynamic data using C# Lists.

---

## 🖥️ How it Works
1. Run the application.
2. Enter the student's name and ID (The system validates the input).
3. Enter 5 grades (The system ensures they are valid numbers).
4. View the formatted report with the final Average.
