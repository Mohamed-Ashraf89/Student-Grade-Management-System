Student Grade Management System 🎓
A robust C# Console Application built as part of the Microsoft: Foundations of Coding Back-End certificate. This project demonstrates core programming concepts and Object-Oriented Programming (OOP) principles.

🚀 Features
Student Profile Creation: Capture student names and unique IDs.

Dynamic Grade Storage: Uses List<float> to store multiple subject grades.

Automated Calculations: Built-in logic to calculate the average grade.

Data Validation: Implements int.TryParse and float.TryParse with loops to ensure the program never crashes due to invalid user input.

🛠️ Concepts Applied
Object-Oriented Programming (OOP): Encapsulation through a dedicated Student class and constructor.

Control Flow: Advanced use of while loops for input verification and for loops for data entry.

Collections: Management of dynamic data using C# Lists.

🖥️ How it Works
Run the application.

Enter the student's name and ID (The system validates that the ID is a valid integer).

Enter 5 grades (The system validates that each grade is a valid number).

The application displays a formatted report showing the Name, ID, and the Average Grade (formatted to 2 decimal places).
